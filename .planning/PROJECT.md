# bac_ses — BAC 2026 SES Study Platform

**Vision:** A focused, no-fuss study companion for preparing the French Baccalauréat SES (Sciences Économiques et Sociales) exam.

**Audience:** Myself (single-user, personal prototype).

**Core problem:** Organize SES course material by topic, test knowledge retention with active recall (quizzes/flashcards), and track progress toward exam readiness — all in one place without the overhead of generic study apps.

**Scope boundary:** Quick prototype. No auth, no multi-user, no backend. Everything runs in the browser via React + local state / localStorage.

**Tech:** Vite 8 + React 19 + TypeScript 6. Single-page app, no router initially, CSS modules or inline styles.
